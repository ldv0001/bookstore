# Bookstore

This is a simple bookstore website example

# Stack

+ ### Spring boot
+ ### Spring security
+ ### PostgreSQL
+ ### React
+ ### Docker


# Building
In back-bookstore/
### Windows
```
mvnw.cmd package
```
### Linux

```
./mvnw package
```
Run all containers:

```
docker-compose up
```
Once started, the application should be available at:

```
http://localhost:3000
```

To sign up as admin:
+ login "admin"
+ password "root"
